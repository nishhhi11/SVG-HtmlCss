
#  CSS & SVG Animation Demo

A simple project demonstrating **CSS Transitions**, **CSS Keyframe Animations**, and **SVG Motion Animations**.

##  Features

###  CSS Transition Box

* Color change on hover
* Scale effect
* Border-radius transformation
* Shadow & opacity effect
* Smooth 0.3s transitions

###  CSS Animation Box

* Moves horizontally
* Rotates 360°
* Scales up
* Changes color & opacity
* Infinite alternate animation

###  SVG Animation

* Circle follows a curved motion path
* Size animation
* Color transition (blue → red)
* Opacity animation
* Continuous rotation

##  Technologies Used

* HTML5
* CSS3
* SVG

---

**Created for learning CSS & animation fundamentals.**
